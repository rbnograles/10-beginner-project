# 10 Beginner Project Using HTML CSS & JS

## Description

This repository contains 10 beginner projects made using html css and javascript to help newbie web developer further enhance their skills.

## Features

- Todo List App
- Calculator App
- Weather App
- Countdown Timer
- Quiz App
- Stopwatch App
- BMI Calculator
- Currency Converter
- Recipe App
- Random Quote Generator

## Technologies

- HTML
- CSS
- JavaScript
